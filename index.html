Featured Projects

TCRi

Information-Theoretic Framework for T-Cell Analysis

Nature Genetics (under review)

Built a hierarchical Bayesian framework with VampPrior VAE to infer joint clonotype-phenotype distributions in single-cell T-cell data. The model combines variational inference with mixture-of-Dirichlet priors to learn interpretable latent representations. Information-theoretic metrics (mutual information, entropy) derived from learned distributions predict immunotherapy response across multiple cancer types.

PyTorch Pyro Variational Inference Hierarchical Models Information Theory

GeneVector

Dense Vector Representations via Mutual Information

Nature Communications 2023

Developed an information-theoretic embedding framework that adapts skip-gram objectives to gene co-expression structure. The model computes mutual information between gene expression patterns to create interpretable vector representations, enabling compositional reasoning through latent space arithmetic. Framework scales to 250K+ cells while maintaining interpretability and has been used across multiple published studies.

PyTorch Skip-gram Mutual Information Representation Learning Vector Space Models

Grafiti

Graph Attention for Spatial Representation Learning

In Preparation

Designed a multi-task graph attention framework with dual decoders for joint spatial graph and gene expression reconstruction. Implemented orthogonal subspace separation to decouple spatial context from cell identity features. Tensorized DEC clustering head with low-rank affinity coupling produces interpretable joint distributions for information-theoretic motif discovery. System uses GradNorm for automatic task balancing and supports multi-GPU training.

CircadiOmics

Circadian Omic Web Portal

Nucleic Acids Research 2018

Developed a comprehensive web platform integrating 227 circadian omic datasets across eight species. Implemented graph-based data models linking oscillatory patterns across transcriptomic, metabolomic, and proteomic datasets. Supports interactive visualizations, comparative analyses, and link prediction to identify potential circadian regulators. Applied Gaussian process models for oscillation pattern inference.
